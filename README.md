# Theodorus Leander - Software Engineer Portfolio

Selamat datang di repositori website portofolio pribadi saya. Website ini dirancang untuk menampilkan perjalanan profesional, proyek pilihan, dan keahlian teknis saya di bidang rekayasa perangkat lunak.

## Tentang Saya
Saya adalah alumnus Rekayasa Perangkat Lunak dari SMK Strada Jakarta. Saya memiliki minat besar dan fokus karir pada Backend Development, AI Engineering, serta Game Development. 

Melalui pengalaman magang di industri (seperti di PT Rayterton Indonesia dan PT Dama Integra Solusindo), saya terbiasa memecahkan masalah sistematis, memelihara basis kode, dan merancang arsitektur database yang aman.

## Tech Stack & Keahlian
Website portofolio ini dibangun dengan:
* **Frontend:** Vanilla HTML5, CSS3, JavaScript (ES6+)
* **Animations:** GSAP & ScrollTrigger
* **Deployment:** Firebase Hosting

**Keahlian Utama (Core Competencies):**
* **Backend:** PHP (Native & Laravel), MySQL, REST API Engineering. (Berpengalaman merestrukturisasi sistem backend skala enterprise dengan PHP Native).
* **Game & AI Logic:** Unity Engine, C# (State machine & mekanika interaktif).
* **Frontend & Mobile:** HTML, CSS (Tailwind/Bootstrap), JavaScript, React Native[cite: 1].
